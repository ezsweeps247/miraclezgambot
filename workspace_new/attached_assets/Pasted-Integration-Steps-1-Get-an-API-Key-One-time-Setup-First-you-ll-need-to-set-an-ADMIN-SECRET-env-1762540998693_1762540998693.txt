Integration Steps
1. Get an API Key (One-time Setup)
First, you'll need to set an ADMIN_SECRET environment variable and generate an API key:

# Set admin secret (do this once in your deployment settings)
ADMIN_SECRET=your_strong_secret_here
# Then call the admin endpoint to create an API key
curl -X POST https://your-fundora-blox-domain.com/api/admin/keys \
  -H "X-Admin-Secret: your_strong_secret_here" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "My Gaming Platform",
    "webhookUrl": "https://yourplatform.com/webhooks/fundora",
    "webhookSecret": "webhook_secret_for_verification"
  }'

You'll get back an API key like fblox_live_abc123... - save this securely!

2. Create a Game Session (When Player Wants to Play)
When a player wants to play, create a session:

const response = await fetch('https://your-fundora-blox-domain.com/api/game/sessions', {
  method: 'POST',
  headers: {
    'X-API-Key': 'fblox_live_abc123...',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    externalPlayerId: 'user_12345',      // Your user ID
    playerName: 'JohnDoe',
    initialCredits: 100.00,              // Starting balance
    stake: '1'                           // Game stake ($1)
  })
});
const { session, embedUrl } = await response.json();

3. Embed the Game in Your App
Use the embedUrl you received:

<iframe 
  src="https://your-fundora-blox-domain.com?session=SESSION_TOKEN" 
  width="100%" 
  height="800px"
  frameborder="0"
  allow="fullscreen">
</iframe>

4. Manage Player Credits
// Load credits (when player deposits money)
await fetch('https://your-fundora-blox-domain.com/api/game/credits/load', {
  method: 'POST',
  headers: {
    'X-API-Key': 'fblox_live_abc123...',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    externalPlayerId: 'user_12345',
    amount: 50.00,
    reference: 'DEPOSIT_ORDER_789'
  })
});
// Check balance
const balance = await fetch(
  'https://your-fundora-blox-domain.com/api/game/credits/balance/user_12345',
  { headers: { 'X-API-Key': 'fblox_live_abc123...' } }
);
// Redeem credits (when player withdraws)
await fetch('https://your-fundora-blox-domain.com/api/game/credits/redeem', {
  method: 'POST',
  headers: {
    'X-API-Key': 'fblox_live_abc123...',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    externalPlayerId: 'user_12345',
    amount: 25.00,
    reference: 'WITHDRAWAL_REQ_123'
  })
});

5. Handle Webhooks (Optional but Recommended)
Set up an endpoint in your app to receive real-time game events:

// Your webhook endpoint: https://yourplatform.com/webhooks/fundora
app.post('/webhooks/fundora', (req, res) => {
  const signature = req.headers['x-webhook-signature'];
  const eventType = req.headers['x-event-type'];
  
  // Verify signature for security
  const crypto = require('crypto');
  const hash = crypto
    .createHmac('sha256', 'webhook_secret_for_verification')
    .update(JSON.stringify(req.body))
    .digest('hex');
  
  if (hash === signature) {
    // Handle different event types
    if (eventType === 'game.ended') {
      const { externalPlayerId, score, prize } = req.body;
      // Update your database, notify user, etc.
    }
  }
  
  res.sendStatus(200);
});
